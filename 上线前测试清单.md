# 小码快改上线前测试清单

## 测试目标

确认第一版 MVP 可以稳定完成核心闭环：上传 HTML → 编辑内容 → 预览 → 导出 → 本地重新打开检查。

建议至少准备 10-20 个真实 HTML 文件，其中包含普通网页 HTML 和飞象课件 HTML。

## 基础打开

- [ ] 直接使用版解压后双击 `index.html` 可以正常打开。
- [ ] 部署版上传到静态网站后可以正常打开。
- [ ] 顶部 Logo、按钮、画布、属性面板显示正常。
- [ ] 示例页面可以正常载入。

## 上传识别

- [ ] 可以上传 `.html` 文件。
- [ ] 可以上传 `.htm` 文件。
- [ ] 非 HTML 文件会提示“请选择 HTML 文件”。
- [ ] 普通 HTML 能正常进入编辑模式。
- [ ] 飞象课件 HTML 能正常识别并进入课件编辑模式。
- [ ] 大文件上传后页面没有明显卡死。

## 文字编辑

- [ ] 点击文字后属性面板显示文本内容。
- [ ] 修改整段文字后画布内容同步变化。
- [ ] 只选中部分文字时，修改字体只影响选中片段。
- [ ] 只选中部分文字时，修改颜色只影响选中片段。
- [ ] 只选中部分文字时，修改加粗只影响选中片段。
- [ ] 未选中文字、只点击整个文字框时，样式作用于整个文字框。

## 图片编辑

- [ ] 选中图片后可以修改图片说明。
- [ ] 可以替换本地图片。
- [ ] 替换后的图片能显示。
- [ ] 导出后替换图片仍然存在。
- [ ] 图片圆角可以调整。
- [ ] 图片宽高可以通过属性面板调整。

## 模块编辑

- [ ] 修改模块背景色时，模块大小不发生突变。
- [ ] 修改模块圆角正常生效。
- [ ] 修改模块宽度正常生效。
- [ ] 修改模块高度正常生效。
- [ ] 右侧拖拉点只改变宽度。
- [ ] 左侧拖拉点向左拉时右边位置保持不动。
- [ ] 上方拖拉点向上拉时底部位置保持不动。
- [ ] 下方拖拉点只向下改变高度。
- [ ] 四角拖拉点保持比例缩放。
- [ ] 模块可以通过绿色拖拽手柄移动位置。

## 表格编辑

- [ ] 点击表格单元格后显示表格工具。
- [ ] 可以增加行。
- [ ] 可以删除行。
- [ ] 只有一行时不允许继续删除。
- [ ] 可以增加列。
- [ ] 可以删除列。
- [ ] 只有一列时不允许继续删除。
- [ ] 可以选中整行。
- [ ] 可以选中整列。
- [ ] 选中整列后修改宽度，整列同步变化。
- [ ] 选中整行后修改高度，整行同步变化。

## 撤销恢复

- [ ] 修改文字后可以撤销。
- [ ] 修改样式后可以撤销。
- [ ] 删除元素后可以撤销。
- [ ] 撤销后可以恢复。
- [ ] 最多支持撤销 5 步。
- [ ] 超过 5 步继续撤销时提示“最多只能撤5步”。
- [ ] 撤销后画布滚动位置不回到顶部。

## 预览导出

- [ ] 点击预览可以看到当前编辑结果。
- [ ] 关闭预览后回到编辑界面。
- [ ] 点击导出可以下载 HTML。
- [ ] 导出的 HTML 可以在浏览器中打开。
- [ ] 导出的普通 HTML 样式基本保持。
- [ ] 导出的飞象课件 HTML 页面结构基本保持。
- [ ] 导出文件中没有编辑器辅助控制点。

## 反馈与隐私

- [ ] 点击“反馈问题”能打开说明弹窗。
- [ ] 点击“隐私说明”能打开说明弹窗。
- [ ] 弹窗可以正常关闭。
- [ ] 隐私说明与当前产品行为一致：默认本地处理，不主动上传文件。

## 埋点检查

- [ ] 打开浏览器控制台，操作上传、编辑、预览、导出时能看到 `[小码快改埋点]` 日志。
- [ ] 本地存储 `xiaoma_kuaigai_events` 中有最近事件记录。
- [ ] 后续接入正式统计工具前，不记录用户文件正文内容。

## 封版标准

- [ ] 10 个以上真实 HTML 文件测试通过。
- [ ] 3 个以上真实飞象课件文件测试通过。
- [ ] 上传、编辑、导出主链路无阻塞问题。
- [ ] 已知问题均记录到内测反馈表。
- [ ] 可以发给第一批内测用户试用。
